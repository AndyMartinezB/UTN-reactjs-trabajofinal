# Trabajo Final Integrador

## Desarrollo en React JS​: Construcción de una Aplicación Web con Autenticación y Dashboard

## Objetivo

Desarrollar una aplicación web utilizando React JS que integre autenticación de
usuarios, navegación entre páginas y gestión de datos, aplicando los conceptos
de componentes, estados, props, contexto global y routing vistos a lo largo del
cursado.
